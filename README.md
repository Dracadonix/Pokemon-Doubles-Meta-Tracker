## 🚀 How It Works
1. GitHub Actions fetches Smogon 2026 data every hour
2. Data is saved to `meta_data.json`
3. GitHub Pages serves the website
4. Your browser displays live 2026 meta data

## 🛠️ Setup
1. Fork this repository
2. Enable GitHub Pages (Settings → Pages → main branch)
3. Watch it auto-update with 2026 data!
